百度旅游app简析

Overview

最近对旅游类app较感兴趣，毕竟旅游这个主题是和移动天生契合的。大致扫完现有app，比较关注三个细分方向：出行推荐，个人旅游日志，图片社交。其实还有很多其他细分，最重要之一是旅游信息的提供，但我想信息源的数量和质量，恐怕百度等大公司已做得足够好。

Btw，其实大多app都是综合性闭环（即推荐，攻略，住宿，图片or简单游记等，集多功能于一体），只不过会强调突出其中的一两项功能。也不排除有很多精美的单一功能向app，比如禅游记。

第一个旅游app简析，选择了百度旅游。

百度旅游

首页视觉效果超赞，渐变背景色+透明layout+嵌入本地美景图，顶部两个button也非常干净。不过广告是败笔，整个UI档次瞬间下滑一档。

主浅蓝到绿的渐变，奠定主线，明亮，适当轻快，给人以放松的感受。

功能元素：

第一层级：本地旅游 + 推荐目的地 + 灵感（Just like “I’m feeling lucky”） + 我的旅行

细分层级：

1. 本地旅游

  • 游玩指南

综合性指南：覆盖推荐功能以及出行入住的信息。
特色景点推荐 + 特色餐厅推荐 + 门票优惠 + 酒店预订 + 周边游。

依然是较淡的背景色调，用metro块做分类，其中嵌入圆形图标，加强对比度显示。顶部button和底部tab/Actionbar，采用蓝色与白色衬底与首页呼应。

  • 现场直播

同地驴友分享图片，以时间/直播作为互动元素。

小瀑布式展示 感觉尚好，但是驴友的分享内容是否存有很好的吸引力，存疑。

  • 我的附近

根据当前地理位置推荐景点 餐馆 酒店 娱乐信息

  • 我的行程

2. 推荐目的地

进行关系分类，然后对不同类别进行景点推荐，简单有效。

所分类为：情侣 自驾 美食 境外热门 单身 四月深呼吸

注意：分类的最后一类是根据当前月份进行的推荐。

3. 灵感/I’m feeling lucky

触感很像google的手气不错，表现形式和人人的新鲜事很像。半瀑布流展示所有细分，如热门精选（地点推荐），新鲜事（活动推荐），明信片（图片分享），这个层级把握很好，不会让我觉得是杂烩，即使找不到自己想要的信息源，也可以带给我欣赏的快乐。



总体感受

  • 一站式多功能，重点在提供‘玩在当地’的旅游信息（比如活动，住宿，特色景点/餐厅），信息源丰富，功能多而强。
  • 轻推荐，轻游记，轻图片社交。做得简单明了。
  • 灵感/I’m feeling lucky 是个小小的体验高潮点
